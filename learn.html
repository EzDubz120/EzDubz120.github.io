Web Development

HTML stands for hyper text markup language and is used as the basic framework for most websites, and to create a website html is a great place to start

It is easy to learn, has very basic syntax and uses different tags for example the "<p>" tag is used to define a paragraph.

CSS stands for cascading style sheets and is used for the graphical elements of a website for example some basic CSS uses are: Text colour, font and size.

PHP is used to add functionality to a website without the need for complex servers and files

JavaScript is a very versatile language. It can be used for front and back end and is the most commonly used language for web development

SQL stands for structured Query Language and is used for data management in databases

Commonly used terms

Front end - the visual side of the website

Back end - the server side of the website

Full stack developer - A developer who programs the front and back end

IDE - stands for 'integrated developer enviroment', this is a code editor with more features like debugging and numerous other developer features an example of this is Visual studio and visual studio code.

Basic HTML tags

All HTML tags have a start tag and an end tag, a start tag looks like this: "<p>" and an end tag looks like this: </p>

"<p>" - This tag denotes a paragraph, just like this text here.

"<h1>" - This tag denotes a heading, the h means heading and the number denotes a size, the sizes are already set, from 1 to 6 with 1 being the biggest and 6 being the smallest, however these sizes can be changed

"<DOCTYPE>" - denotes the document type, for example HTML

"<b>" - denotes whether text should be bold or not

"<u>" - denotes whether text should be underlined or not

"<body>" - states the documents body: note there can only be 1 of these tags per document

"<form>" - denotes a form in the HTML document

Basic Programming operators

>= - greater than or equal to

|| - used in some languages as OR

&& - used in some languages as AND
