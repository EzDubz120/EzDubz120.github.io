Learn HTML and CSS

HTML stands for hyper text markup language and is used as the basic framework for most websites, and to create a website html is a great place to start

It is easy to learn, has very basic syntax and uses different tags for example the "< p >" tag is used to define a paragraph.

CSS

CSS stands for cascading spread style and is used for the graphical elements of a website for example some basic CSS uses are: Text colour, font and size.

PHP

PHP is used to add functionality to a website without the need for complex servers and files

JavaScript is a very versatile language. It can be used for front and back end and is the most commonly used language for web development

Commonly used terms

Front end - the visual side of the website

Back end - the server side of the website

Full stack developer - A developer who programmes the front and back end
